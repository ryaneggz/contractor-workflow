# WhatsApp Message Templates for Crew

## Job Offer

```
Hey {crew_name}, got a new job available:

{project_name}
Address: {address}
Scope: {scope_summary}
Pay: ${crew_pay}
Timeline: {start_date} - {end_date}

Reply YES to accept or NO to pass.
```

## Job Offer Reminder (24h no response)

```
Hey {crew_name}, following up on the {project_name} job.

Pay: ${crew_pay}
Timeline: {start_date} - {end_date}

Still interested? Reply YES or NO.
```

## Job Confirmed

```
You're on for {project_name}.

Address: {address}
Start: {start_date}
Materials budget: ${materials_budget}

Send me photos of receipts as you buy materials. I'll track everything.
```

## Daily Progress Check-in

```
How's {project_name} going? Any updates or receipts to share today?
```

## Receipt Acknowledged

```
Got it. Receipt from {store} for ${amount} logged.

Materials budget remaining: ${remaining} of ${materials_budget}
```

## Budget Warning (< 10% remaining)

```
Heads up on {project_name}: only ${remaining} left in the ${materials_budget} materials budget.

Check with {contractor_name} before making more purchases.
```

## Budget Exceeded

```
{project_name} materials are ${overage} over the ${materials_budget} budget.

Hold off on purchases and check with {contractor_name}.
```

## Project Completion Confirmation

```
Thanks for wrapping up {project_name}!

Total materials tracked: ${total_materials}
Your pay: ${crew_pay}

I'll send the invoice to the client and get you paid.
```
